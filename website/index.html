How It Works

24 URL features analyzed by a trained Random Forest classifier

01

Feature Extraction

The URL is parsed and 24 lexical features are extracted — including length, special characters, domain properties, and suspicious patterns.

→
02

ML Inference

Features are scaled and fed into a trained Random Forest model. The model outputs class probabilities for each URL.

→
03

Risk Assessment

Probabilities are mapped to CRITICAL / HIGH / MEDIUM / LOW risk levels, and a full report is returned in milliseconds.
